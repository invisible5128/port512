Bible In A Year App

iOS App which helps users to plan and read Bible in an Year.

About Bible In A Year

Bible In A Year is an iOS app built using SwiftUI, developed and published to App Store. It helps users to track their daily progress as they start their journey in reading Bible In A Year.

App Store link: App Store

Github link to the project: Github Link

APP features

  • Daily progress tracking with SwiftData for local storage.
  • Integrated Bible for seamless in-app reading.
  • Algorithmically generated 365-day reading plan to ensure structured progress.
  • User-friendly UI/UX with smooth navigation.
